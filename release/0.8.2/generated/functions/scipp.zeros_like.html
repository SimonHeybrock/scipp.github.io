scipp.zeros_like¶

scipp.zeros_like(var)¶

Constructs a Variable with the same dims, shape, unit and dtype as the input variable, but with all values initialized to 0. If the input has variances, all variances in the output are set to 0.

Seealso

scipp.zeros() scipp.ones_like()

Parameters

var (scipp._scipp.core.Variable) – Input variable.

Return type

scipp._scipp.core.Variable
