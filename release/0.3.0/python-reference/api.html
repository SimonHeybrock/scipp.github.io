Classes¶

Dim

Dimension label

Unit

A physical unit.

Variable

Array of values with dimension labels and a unit, optionally including an array of variances.

VariableView

View for Variable, representing a sliced or transposed view onto a variable; Mostly equivalent to Variable, see there for details.

DataArray

Named variable with associated coords, masks, and attributes.

DataArrayView

View for DataArray, representing a sliced view onto a DataArray, or an item of a Dataset; Mostly equivalent to DataArray, see there for details.

Dataset

Dict of data arrays with aligned dimensions.

DatasetView

View for Dataset, representing a sliced view onto a Dataset; Mostly equivalent to Dataset, see there for details.

GroupByDataArray

GroupBy object implementing to split-apply-combine mechanism.

GroupByDataset

GroupBy object implementing to split-apply-combine mechanism.

Counts¶

“Counts” refers to the default unit of data after histogramming. It is also known as frequency. This can be converted into a counts (frequency) density, e.g., for visualization purposes.
